## Project Scope

**Nguồn dẫn:** [https://github.com/Tson28/Fashion-store-management-master.git](https://github.com/Tson28/Fashion-store-management-master.git)

Phạm vi của dự án

- Đăng nhập:
  - Quên mật khẩu.
- Hàng hóa:
  - Quản lý sản phẩm
  - Quản lý mặt hàng.
  - Quản lý thuộc tính sản phẩm.
  - Quản lý nhà cung cấp.
- Giao dịch:
  - Nhập hàng.
  - Bán hàng.
  - Trả hàng.
  - Đổi hàng.
  - Hóa đơn bán hàng.
  - Hóa đơn nhập hàng.
  - Hóa đơn trả hàng.
  - Hóa đơn đổi hàng
- Thống kê:
  - Thống kê doanh thu.
  - Thống kê doanh số.
- Nhân viên:
  - Quản lý nhân viên.
  - Lương.
- Khách hàng: Quản lý khách hàng
- Cá Nhân: Quản lý thông tin cá nhân
